Web Addressing

Links and URLS

Related Hypertext (HTML) documents can be navigated through links in pages.

Links are usually shown as descriptive text, and link to some web source.

The "location" of the source is a Uniform Resource Locator.

Scheme

The scheme part of the URL indicates the format in which the request should be sent.

Commonly supported ones are:

  • ftp://
  • http://
  • https://
  • file://
  • mailto://

Host Names

Hostnames can be local example 'localhost' and also global example 'port.ac.uk' Every computer on the internet has a unique IP address. Global internet hostnames are translated to IP addressing using the Domain Name System (DNS). Due to this you can also use the IP address in the URL instead of the hostname

Domain Name Service (DNS)

This is a global record of domains to IP addresses. Until recently, domains were allocated under countries.

  • .com ← US or International
  • .uk ← UK, previously restricted to third level:
    • .co.uk
    • .ac.uk
    • .nhs.uk

Graphical Interchange Format (GIF)

used on websites for low frame rate animated videos. As time progressed GIFs have started to use higher frame rates.

  • 256 Colours (Selected out of 16 Million)
  • 1 Transparent background colour

Portable Network Graph (PNG)

  • Lossless 16 Million (24 - bit) colours
  • 8 - bit transparency

Joint Photographic Experts Group (JPEG)

  • For photographs, smooth images, lossy
  • No transparency

Standards

Introduction

Standards are a way of making sure that same type of components will work with each other.

Examples

There are many different examples of standardization around the world. Here are some examples:

  • Plugs no matter where in the UK will work with plug sockets.
  • Measurements will be the same throughout the world
  • Protocols on computers will be the same to allow for communication between computers.

Why have standards?

Standards allow for new inventions to work with current inventions. An example of this can be if a website was going to be created they know what the standards are for the internet. They can create the website to correctly work with the current protocols. This will mean that they will be able to make the website work on all different types of computers.
